# create-ssam

Quicstart a sketch with [`ssam`](https://github.com/cdaein/ssam).

## How to

```sh
npm create ssam
```

Then, follow the prompts.

## Templates

- Vanilla
  - TypeScript
  - JavaScript
- OGL
  - Full Screen Shader TS
- more will be added...
