# create-ssam

Quicstart a sketch with [`ssam`](https://github.com/cdaein/ssam).

## How to

```sh
npm create ssam
```

Then, follow the prompts.

To update to the latest package,

```sh
 npm create ssam@latest
```

## Templates

- Vanilla
  - TypeScript
  - JavaScript
- OGL
  - Fullscreen Shader TS
  - Basic Cube TS
- Three
  - Fullscreen Shader TS
  - Fullscreen Shader JS
- more will be added...
